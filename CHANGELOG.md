## 0.0.1 — 2025-11-13

- Первая публичная версия CommentCorrector для VS Code.
- Автоматическое исправление неформальных комментариев в корпоративный стиль.
- Подсветка формальных и неформальных комментариев с возможностью сброса выделения.
- Интеграция с OpenAI-совместимыми API для обработки текста. Поддерживаются любые модели, совместимые с Chat Completions API (OpenAI, OpenRouter, GitHub Models, локальные серверы и др.).
